1. 필요한 페이지 목록
- 메인 페이지
- 연극 페이지(테이블)
- 뮤지컬 페이지(테이블)
- 리뷰 페이지(리스트)
- 관리자 페이지 & 마이 페이지

    > 관리자 페이지
    
    관리자로 로그인되어 있을 때 마이 페이지를 로딩하면 관리자 페이지가 출력된다
    
     ![1](./image/1.png)
    
    관리자 페이지에서는 연극과 뮤지컬을 업로드할 수 있는 버튼이 있다
    
     ![2](./image/2.png)
    
    버튼을 클릭하면 폼이 출력되며 그 폼은 포스터 이미지, 제목, 연극인지 뮤지컬인지 구분할 수 있도록 
    입력하는 칸이 존재
    form은 enctype='multipart/formdata'
